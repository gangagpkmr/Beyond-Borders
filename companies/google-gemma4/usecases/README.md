# Google Gemma 4 - Use Cases & Problem Statements

This folder contains use cases, problem statements, and target opportunities mapping the Google Gemma model family to solve real-world problems.

---

## Current Use Cases

*Currently open for community contributions.*

To add a use case:
1. Review the [Evaluation Framework](../../../frameworks/evaluation-framework.md) and [Edge AI Principles](../../../frameworks/edge-ai-principles.md).
2. Copy the [Use Case Template](../../../usecase-template.md) and save it in this directory (or in a vertical subdirectory like `healthcare-ai/` or `agriculture-ai/`).
3. Fill in the template details including Desirability, Feasibility, Viability, and the Startup Potential Score.
4. Name the file using lowercase kebab-case (e.g. `offline-crop-consultant-llm.md`).
5. Open a Pull Request.
