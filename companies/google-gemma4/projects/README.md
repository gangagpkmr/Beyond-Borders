# Google Gemma 4 - Open Projects

This folder houses open-source projects, tool integrations, and prototype codebases built around the Google Gemma model family.

---

## Current Projects

*Currently open for community contributions.*

To add a project:
1. Copy the [Project Template](../../../project-template.md) and save it in this directory.
2. Complete the sections detailing your Technical Approach, Milestones, and required developer skills.
3. Link your project to any relevant problem statement in the `usecases/` folder.
4. Name the file using lowercase kebab-case (e.g. `gemma-offline-translation-engine.md`).
5. Open a Pull Request.
