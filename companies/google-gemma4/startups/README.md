# Google Gemma 4 - Startup Opportunities

This folder lists validated startup opportunities, business models, and commercialization maps enabled by Google Gemma models.

---

## Current Opportunities

*Currently open for community contributions.*

To add a startup opportunity:
1. Review the [Startup Readiness Score (SRS) Rubric](../../../frameworks/startup-readiness-score.md).
2. Copy the [Startup Template](../../../startup-template.md) and save it in this directory.
3. Fill in the pitch, business model, market opportunity (TAM/SAM), and your calculated SRS score.
4. Name the file using lowercase kebab-case (e.g. `gemma-powered-rural-tutor.md`).
5. Open a Pull Request.
