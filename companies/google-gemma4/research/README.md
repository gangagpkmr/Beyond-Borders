## Current Research Topics

*Currently open for community contributions.*

To add a research direction:
1. Document the core research gap or hypothesis (e.g., performance of quantized PaliGemma models on specialized thermal datasets, or optimizations for low-power edge compute).
2. Detail the methodology, expected experiments, and key datasets.
3. Link to any corresponding files in `usecases/` or `projects/` if applicable.
4. Name the file using lowercase kebab-case (e.g. `gemma-inference-quantization-loss-analysis.md`).
5. Open a Pull Request.
